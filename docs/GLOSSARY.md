# GLOSSARY.md — 語句リスト(図に表示される用語の統一)

このファイルは、`scripts/run_*.py` が生成する **人に見せる図 (results/<シナリオ>/*.png, *.gif, *.mp4)** に
表示されるすべての語句を一覧化し、表記・意味・対応英語を統一するためのものである。
発表資料・README・コード・図のラベルで語句がぶれないよう、ここを正準とする。

- 各図は日本語フォントが使える環境では **日本語ラベル**、無い環境では **英語ラベル**を出す
  (`Lbl(ja, en)` / `L(ja, en)`)。本リストは「日本語表記 / 英語表記」を併記する。
- 記号(`d`, `σ_ang`, `φ`, `B` など)の正準定義は `docs/MATH_SPEC.md` に従う。本リストは
  「図上での表示」を補足するものであり、数式定義を変える場合は MATH_SPEC が優先。

## 1. 座標系・軸 (Coordinate axes)

ENU 系(East-North-Up)。親機を基準とする。

| 図の表記 (日本語) | 図の表記 (英語) | 意味 |
|---|---|---|
| `X [m]` | `X (East) [m]` | 東方向 [m] |
| `Y [m]` | `Y (North) [m]` | 北方向 [m] |
| `Z [m]` | `Z (Up) [m]` | 鉛直上方向 [m](水面=0、水中は負) |

## 2. 登場物 (Entities)

| 図の表記 (日本語) | 図の表記 (英語) | 意味 |

## 3. センサ (Sensors)

`src/results_io.py` の `SENSOR_NAMES` が正準。図・README の凡例はこれに揃える。

| 正準名 (短縮) | 図/README 表記 | 意味 |
|---|---|---|
| 親機カメラ | 親機カメラ (光学角度: 方位/仰角) | 子機の方位・仰角(角度)を測る光学センサ |
| 音響1点 | 音響測距 (親機1点までの距離) | 親機1点までの距離を測る音響センサ |

| 深度 | 深度センサ (圧力→鉛直 z) | 圧力から鉛直 z を直接測る |

## 4. 真値・推定・モード (Truth / Estimate / Modes)

MBD 上、**真値 (truth)** と **推定 (estimate)** は厳密に区別する(図でも色分け)。

| 図の表記 (日本語) | 図の表記 (英語) | 意味 |
|---|---|---|
| 真値 | Truth | 仮想世界の真の位置(評価のときだけ突き合わせ) |
| 真の軌道 | true path / true | 真の移動経路 |
| 真の表面 / 真の表面点 | true surface / true | キューブ等の真の表面点群 |
| 推定 / 推定軌道 | estimate / estimate path | 観測のみから推定した位置・経路 |
| 現在の推定 | current | 反復最適化の途中の推定値 |
| 初期値 | Initial | 最適化の初期解 |
| ステレオ推定点群 / ステレオ推定 (×Nフレーム) | stereo estimate | 子機ステレオで三角測量した推定点群 |

### 推定の手法・拘束 (Estimation methods)

| 図の表記 (日本語) | 図の表記 (英語) | 意味 |
|---|---|---|
| 観測のみ | obs only | 観測値だけで推定(拘束なし) |
| 観測 + IMU 拘束 / IMUあり / IMUなし | obs + IMU / with-IMU / no-IMU | IMU の時刻間変位を拘束に加える/加えない |
| 純L2 (linear) | linear | L2 損失の最小二乗(外れ値に弱い) |
| ロバスト (huber / soft_l1 / cauchy) | robust | ロバスト損失。外れ値の残差を減衰 |

### 運用モード・切替 (Operating modes)

| 図の表記 (日本語) | 図の表記 (英語) | 意味 |
|---|---|---|
| 光学モード / 光学 | optical | 親機カメラ(角度)+音響を使うモード |
| フォールバック (距離+IMU+深度) | fallback | 光学なし。距離+IMU+深度で測位 |
| 運用不可 | none | 目標精度を満たせない領域 |
| 光学なし推定 / 光学なし (距離+IMU+深度) | no-optical | 光を使わない測位(濁り非依存) |
| 光学あり coastal | optical coastal | 比較用:濁り coastal での光学測位 |
| 単独 (音響1点距離+IMU+深) | single range | SBL でなく音響1点+IMU+深度 |
| SBL推定 / SBL (音響4点距離+IMU+深) | SBL / SBL (4 ranges) | SBL 多辺測量による測位 |
| 深度なし / 深度あり | no depth / with depth | 深度センサ融合の有無 |
| 切替境界 (見失いN%) / 自動切替 / 光学使用 (自動切替) | optical used | 光学↔フォールバックの自動切替境界・使用状況 |

## 注記

- 図のラベルは ASCII/日本語の自動切替のため、**英語表記も同時に統一対象**とする。
- 新しい図・語句を追加したら、必ずこのリストに追記する(語句のぶれを防ぐ)。
- センサ名は `src/results_io.py::SENSOR_NAMES`、記号は `docs/MATH_SPEC.md` が単一の出典。
  齟齬を見つけたら、それぞれの出典を直し、本リストを合わせる。
